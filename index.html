Notepad++ PythonScript for Python 3

      
Pyhon 3 용 PythonScript 3는 개발 단계. 기존 Python 2용 PythonScript 3와 같이 사용 불가능.

    
UTF-8 인코딩을 사용한 문서만 지원

      
 64비트 시스템에서 32비트용인 Python 3용 PythonScript 3 사용 불가능.

      
(Python 3.9부터 32비트 시스템 지원하지 않음)

      
현재 Python 3.9 이상 버전도 32비트 시스템 지원함
